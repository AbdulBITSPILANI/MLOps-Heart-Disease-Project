# Screenshots Directory

This directory contains screenshots for the project report.

## Required Screenshots

Please add the following screenshots to this directory:

1. **Class Distribution** (`class_distribution.png`)
   - From EDA notebook showing class balance

2. **Feature Distributions** (`feature_distributions.png`)
   - Histograms of numeric features

3. **Correlation Heatmap** (`correlation_heatmap.png`)
   - Feature correlation matrix visualization

4. **MLflow UI** (`mlflow_experiments.png`)
   - Screenshot of MLflow experiment tracking interface

5. **GitHub Actions** (`github_actions_pipeline.png`)
   - CI/CD pipeline execution screenshot

6. **Docker Build** (`docker_build.png`)
   - Docker image build output

7. **Kubernetes Deployment** (`k8s_deployment.png`)
   - kubectl get outputs showing deployment status

8. **API Testing** (`api_testing.png`)
   - API endpoint testing results

9. **Monitoring Metrics** (`metrics_dashboard.png`)
   - Prometheus metrics or monitoring dashboard

## Note

Screenshots are ignored in git by default. To include specific screenshots:
- Use `git add -f screenshots/filename.png`
- Or modify `.gitignore` to allow specific files
